# PulseApp — Project Overview

## App Identity
- **Name**: PulseApp
- **Type**: Functional mock/prototype — logged-in business management app
- **Industry**: Solo personal training
- **Target User**: A single personal trainer managing their own client base

## User Profile
- **Name**: Sipho Dlamini
- **Business**: Sipho Dlamini | Personal Training
- **Location**: Cape Town, South Africa
- **Active Clients**: 24
- **Monthly Revenue**: ~R18,500

## Tech Stack
- **Framework**: Next.js App Router (v16)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4.1
- **Components**: Untitled UI + React Aria Components
- **Icons**: @untitledui/icons
- **Fonts**: Rubik (Google Fonts)
- **Package Manager**: Bun
- **Deployment**: Vercel

## Design
- **Theme**: Dark mode only
- **Primary Brand**: #5A4EFF (Electric Blue)
- **Accent Lime**: #E2F4A6
- **Accent Lavender**: #EEA0FF
- **Mobile-first**: Optimised for iPhone 14/15 (390px)

## Features
- Dashboard with KPI cards and revenue charts
- Calendar with day/week views
- Client management with detailed profiles
- In-app messaging
- Online booking flow (public, shareable link)
- Workout programming and exercise library
- Nutrition tracking and meal plans
- Digital intake forms and waivers
- E-commerce store (admin + public storefront)
- Loyalty programme with referral tracking
- Notifications centre
- POS / Payments
- Reports and analytics
- Profile with reviews and business settings
